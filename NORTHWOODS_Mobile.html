01 · NORTHWOODS APP▼

NORTHWOODS is a self-hosted AI portal that runs entirely from a single HTML file. All data is stored in your browser's localStorage — no cloud accounts required.

NEXUS TAB

Your primary chat interface. Select a model from the dropdown (Ollama local, Groq, Claude, or OpenRouter), type your message, and hit Send. Responses stream in real time. You can switch system personas (SAIA · Code · Creative · Research) to change how the AI responds.

ORACLE TAB

Deep research mode. Oracle chains multiple AI calls — it breaks your question into sub-queries, synthesises the results, and presents a structured answer. Best for complex technical or research questions where a single LLM pass isn't enough.

GRAPH TAB

A live visual knowledge graph. Nodes represent ideas, entities, or documents. Click any node to expand its connections, drag to rearrange the layout, and double-click to edit the label. Data persists in localStorage under the key nw_graph_nodes.

DATA TAB

A structured key-value data store. Add records manually, import a JSON file, or export everything. Records can be linked to Graph nodes for cross-referencing. Agent outputs are saved here automatically.

CONFIG TAB

Set your API keys, Ollama host URL, default model, OTA update URL, and sync URL. All keys are stored only in your local device storage and are never sent to any server other than the API you're directly calling.

OTA UPDATES

When your Hermes server is running on the same network, the app can update itself without a reinstall:

  1. Open Config tab
  2. Set Update URL to http://<your-PC-IP>:8642/api/update/version
  3. Tap CHECK UPDATE
  4. If a new version is available it downloads and applies automatically on reload

The version is an MD5 hash of NORTHWOODS_Mobile.html — it changes every time you edit that file.

CROSS-DEVICE SYNC

All nw_* localStorage keys (API keys, settings, graph data) can be synced between your desktop browser and the Android app via Hermes:

  1. On your desktop: open Config → set Sync URL → tap PUSH STATE
  2. On your phone: set same Sync URL → tap PULL STATE
  3. Everything is restored in seconds
02 · HERMES SERVER▼

Hermes is a lightweight Flask server that runs on your PC. It acts as a privacy layer between your apps and AI services — your API keys stay on your machine, never in the cloud.

STARTING HERMES

Double-click start_hermes.bat in your NORTHWOODS folder. A terminal opens. Hermes is ready when you see:

Running on http://0.0.0.0:8642

Or from PowerShell:

cd C:\Users\darkh\Documents\NORTHWOODS python hermes_server.py
WHAT HERMES DOES
  • Routes AI requests to local Ollama models on port 11434
  • Stores and serves sync state for cross-device settings
  • Serves OTA app updates — just edit NORTHWOODS_Mobile.html and the app auto-detects the new version
  • Runs a ReAct agent loop for autonomous multi-step tasks
  • Stores memories, tasks, and goals as JSON files in the Neural_Nexus folder
API ENDPOINTS
GET /health — heartbeat GET /api/status — server + Ollama status GET /api/models — list local Ollama models POST /api/chat — streaming chat proxy GET /api/update/version — returns current HTML version hash GET /api/update/app — serves NORTHWOODS_Mobile.html GET /api/sync — retrieve synced state POST /api/sync — push state from a device POST /api/agent/run — run autonomous agent task GET /api/memory — retrieve memories POST /api/memory — save a memory GET /api/tasks — list tasks GET /api/goals — list goals
REMOTE ACCESS VIA CLOUDFLARE TUNNEL

To reach Hermes from outside your home network (e.g. from your phone on mobile data):

  1. Install Cloudflare Tunnel: winget install cloudflare.cloudflared
  2. Run: cloudflared tunnel --url http://localhost:8642
  3. Copy the generated *.trycloudflare.com URL
  4. Paste it into the app's Update URL and Sync URL fields in Config

The tunnel URL changes each time you restart cloudflared unless you create a named tunnel.

OLLAMA SETUP
  1. Download from ollama.com
  2. Install and start Ollama (it runs as a background service on port 11434)
  3. Pull a model: ollama pull llama3 or ollama pull hermes3
  4. In NORTHWOODS Config, set Ollama Host to http://<your-PC-IP>:11434
  5. Select any ollama: model in the Nexus dropdown
03 · SAIA AGENTS▼

SAIA (Sentient Adaptive Intelligence Architecture) agents run autonomous multi-step tasks on your behalf — file operations, web research, code execution, and scheduling — all orchestrated through Hermes.

STARTING AN AGENT

In the Nexus tab, select a SAIA-capable model, then prefix your message with @agent::

@agent: summarise all .txt files in my Downloads folder @agent: research the latest news on quantum computing and save a summary @agent: write a Python script to rename files by date and run it
AGENT TYPES
  • Research — web search + file search, synthesises sources
  • Code — writes and executes Python or JavaScript, returns output
  • File — organises, renames, converts, or reads files on your PC
  • Schedule — sets recurring tasks (requires Hermes to stay running)
AGENT OUTPUT

Results stream into the Nexus chat and are automatically saved to the Data tab under a timestamped key. You can also ask the agent to save specific results:

@agent: research X and save the result to my knowledge graph
SAFETY GUARDRAILS
  • Agents ask for confirmation before deleting files
  • Network requests outside your local subnet require explicit approval
  • All agent actions are logged to NORTHWOODS/Neural_Nexus/agent_log.json
VIEWING LOGS

Open the Data tab and look for the agent_log key, or read the file directly:

C:\Users\darkh\Documents\NORTHWOODS\Neural_Nexus\agent_log.json
04 · CLAUDE CODE▼

Claude Code is Anthropic's agentic coding CLI. It reads your entire codebase, understands it, and can edit files, run tests, and ship changes autonomously — all from your terminal.

INSTALLATION
npm install -g @anthropic-ai/claude-code

Requires Node.js 18+. Check with node --version. If Node isn't installed: winget install OpenJS.NodeJS

STARTING A SESSION
cd C:\Users\darkh\Documents\NORTHWOODS claude

Claude reads your files and waits for instructions. You can also give one-shot commands:

claude "add a dark mode toggle to NORTHWOODS_Mobile.html" claude "fix the OTA update logic so it handles network timeouts better" claude "rebuild the APK after my edits"
KEY SLASH COMMANDS
/help — list all commands /plan — show what Claude intends to do before doing it /compact — summarise context to save tokens on long sessions /continue — resume the last session /clear — clear conversation history /review — ask Claude to review a specific file
WITH NORTHWOODS

Run Claude Code from your NORTHWOODS folder. Claude can:

  • Edit NORTHWOODS_Mobile.html (new tabs, UI fixes, new features)
  • Add or modify Hermes Flask routes in hermes_server.py
  • Rebuild the Android APK (it knows the gradle commands)
  • Debug Capacitor issues and sync the www/ folder
claude "add a voice transcription tab to NORTHWOODS" claude "make the Graph tab support node colours by category"
MCP PLUGINS

Claude Code supports Model Context Protocol plugins that give it access to external services:

claude mcp add github # GitHub repo access claude mcp add slack # Read/write Slack messages claude mcp add google-drive # Access Google Drive files claude mcp list # Show installed plugins
05 · CLAUDE DESIGN · CANVA AI▼

Canva's AI tools let you generate designs, write copy, and create images without leaving the browser. No design experience needed.

ACCESSING CANVA AI
  1. Go to canva.com and sign in
  2. Click Create a design and choose a format
  3. The AI sidebar appears on the right — click Magic Studio
MAGIC DESIGN

Upload a photo or describe a concept in plain English. Canva instantly generates 8+ layout variations. Pick one and customise it.

Example prompt: "A dark tech poster for an AI app called NORTHWOODS with forest imagery and neon green accents"
MAGIC WRITE

Select any text element → right-click → Magic Write → describe what you want. It rewrites, expands, or summarises any copy on the canvas.

TEXT TO IMAGE

Insert → AI Images → describe your image in detail. Generates directly on the canvas. Tip: add style keywords like cinematic, synthwave, dark forest, bioluminescent.

BRAND KIT
  1. Go to Brand → Brand Kit
  2. Upload your logo, set your brand colours (#0a0e17, #00ff9d, #d4af37)
  3. Add your font (JetBrains Mono or Cinzel)
  4. All future AI designs will pull from your Brand Kit automatically
EXPORTING
File → Download → PNG (transparent background for overlays) File → Download → PDF (print-ready) File → Download → MP4 (animated designs) Share → Publish to web (live shareable URL)
WITH NORTHWOODS

Export Canva designs as PNG and drop them into the NORTHWOODS Graph tab as node images. This creates a visual knowledge map where each node can carry a design asset as its visual identity.

06 · OBSIDIAN▼

Obsidian is a local-first markdown knowledge base with a visual graph view — think of it as a desktop equivalent of the NORTHWOODS Graph tab, but for long-form notes and research.

INSTALLATION
  1. Download from obsidian.md
  2. Install and click Create new vault
  3. Point it at a folder, e.g. C:\Users\darkh\Documents\NORTHWOODS\notes\
CORE CONCEPTS
  • Notes — plain .md files. One idea per note.
  • Links — type [[note name]] to create a bidirectional link between notes. Obsidian automatically builds the reverse link.
  • Graph view — press Ctrl+G to open the full graph of all linked notes. Pan, zoom, and click nodes to jump to any note.
  • Tags — add #tag anywhere in a note to group related ideas.
  • Backlinks — every note shows a panel listing all other notes that link to it.
KEYBOARD SHORTCUTS
Ctrl+N — new note Ctrl+G — open graph view Ctrl+P — command palette (search all commands) Ctrl+[ — navigate back Ctrl+Shift+F — search all notes Ctrl+E — toggle edit / preview mode
RECOMMENDED PLUGINS

Settings → Community plugins → Browse → search by name:

  • Dataview — query your notes like a database: LIST FROM #project WHERE status = "active"
  • Templater — create note templates with dynamic fields and JavaScript snippets
  • Kanban — turn any note into a drag-and-drop task board
  • Excalidraw — draw diagrams and wireframes directly inside notes
  • Obsidian Git — auto-commit and sync your vault to a GitHub repo
WITH NORTHWOODS

Point your Obsidian vault at the same folder Hermes can read. Then from the Nexus tab, ask the AI to create, search, or link notes:

@agent: summarise this conversation as an Obsidian note with backlinks @agent: find all notes tagged #research and create a summary note @agent: create a new note about [topic] linked to my existing [[index]] note

You can also ask Claude Code to automate Obsidian workflows — e.g. auto-generate a daily note from your task list every morning.
