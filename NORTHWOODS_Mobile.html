⬡ NEXUS — Command Center▼

The NEXUS tab is your main command center. It shows live stats, quick actions, the Oracle chat, and system status.

KEY FEATURES
  • 4 stat cards (agents, nodes, projects, revenue)
  • Quick action grid for common tasks
  • Oracle Chat with multi-provider AI
  • System status row (Hermes, Ollama, Sync, OTA)
◈ SAIA — Sovereign AI Agent▼

SAIA manages AI agents that can research, code, schedule, and orchestrate tasks.

AGENTS
  • Research — web & vault search
  • Code — generate & run Python
  • File — read/write Hermes filesystem
  • Market — analyze price data
  • Orchestrator — coordinate multi-agent workflows
MISSION CONTROL

Type a goal and click "Deploy SAIA" to assign a mission to the agent fleet.

⬢ GRAPH — Knowledge Graph▼

D3.js force-directed graph of your knowledge nodes. Nodes are colored by type.

CONTROLS
  • Drag nodes to reposition
  • Scroll/pinch to zoom
  • Click node to inspect
  • Use toolbar: layout, charge, link distance, search
  • Cluster pills to show/hide node types
  • 3D mode, matrix rain, SVG export
⚙ CONFIG — Settings▼

Set API keys, Hermes URL, Ollama model, sync, and OTA settings.

API KEYS
Anthropic: sk-ant-... Groq: gsk_... OpenRouter: sk-or-...
HERMES SERVER

Start with: python hermes_server.py

Default port: 8642

⊞ BUILDER — Build & Deploy▼

Manage NORTHWOODS builds, OTA updates, and APK generation.

OTA FLOW
  • App checks GitHub Pages on startup
  • Downloads new HTML if version changed
  • Reloads automatically
APK BUILD

Requires Hermes running with Capacitor build tools. Click "Build APK" in Builder tab.
